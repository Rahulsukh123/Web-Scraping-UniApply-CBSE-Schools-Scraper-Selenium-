# Web-Scraping-UniApply-CBSE-Schools-Scraper-Selenium-
This Python script uses Selenium WebDriver to scrape CBSE school data from UniApply's Nagpur page. It launches Chrome, navigates to the URL, and extracts details like name, address, fees, classes, board, and link from each listing. Data is saved to 'nagpur_cbse_schools_new.csv'. Handles errors by skipping issues, prints scrape count
